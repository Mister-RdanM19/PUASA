# PUASA-2026

Situs sederhana "Jam Digital & Jadwal Sholat" untuk tampilan lokal.

Isi file:
- `index.html` - halaman utama
- `css/styles.css` - gaya tampilan
- `js/scripts.js` - jam digital, jadwal contoh, dan amalan setelah sholat

Cara menjalankan:
1. Buka `index.html` di browser (cukup buka file lokal atau deploy ke hosting statis).
2. Untuk jadwal sholat dinamis, sesuaikan `js/scripts.js` atau integrasikan API jadwal sholat.

Semoga bermanfaat.
